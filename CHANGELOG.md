# Changelog

## [3.2.0] — 2026-06-16

### Added
- **Категориальные цвета на таймлайне** — точки окрашены по категории места: 🍽 зелёный, 🚶 синий, 👁 оранжевый, 🏨 фиолетовый
- **Toast-уведомления** вместо alert/confirm для неразрушающих операций (добавление/удаление мест, сброс данных)
- **Индикатор загрузки карты** — спиннер при инициализации Яндекс.Карт
- **Подсказка drag & drop** — однократное уведомление «Перетащите места для сортировки» при первом просмотре мест
- **Google Font (Nunito)** для более современной типографики

### Changed
- **Полный редизайн CSS**: новые карточки локаций (цветной акцент слева, подъём при наведении), карточки мест (белый фон с рамкой), переработанный таймлайн (цветные точки, ховеры), модальные окна (анимация, тени), адаптация под Nunito

## [3.2.1] — 2026-06-16

### Added
- **Кнопка 🔗** в карточке места и в панели доступных мест — открывает URL места в новой вкладке (отображается только если у места указана ссылка)
- **Карта маршрута** на странице маршрутов — отображает выбранные места в порядке маршрута с порядковыми номерами на маркерах
- **Favicon** — эмодзи 🏖 во вкладке браузера

## [3.1.0] — 2026-06-16

### Removed
- **Поиск мест через Яндекс.Карты** — удалён неработающий функционал (геокодер возвращал некорректные результаты)

### Added
- **Редактирование и удаление мест** прямо со страницы маршрута (кнопки ✎ и ✕ в панели «Доступные места»)
- **Добавление новых мест** со страницы маршрута (кнопка «+ Место» в заголовке панели «Доступные места»)
- **Поле URL** у мест — можно указать ссылку на Яндекс.Карты или сайт. Отображается в карточке места и в панели доступных мест

### Changed
- Пример данных (`data/example.json`) — Париж и Санкт-Петербург заменены на Кронштадт и Петергоф с новыми местами; добавлены `url` у некоторых мест
- `add-place` и `edit-place` теперь корректно обновляют панель доступных мест на странице маршрута

## [3.0.0] — 2026-06-16

### Added
- **Визуальный таймлайн** с автоматическим расчётом времени начала/окончания для каждого элемента маршрута
- **Настраиваемая длительность** для каждого места (15 мин – 4 ч) через выпадающий список в таймлайне
- **Настраиваемое время старта дня** (`<input type="time">`) для каждого маршрута
- **Отель в маршруте** — специальный элемент `__hotel__` можно добавлять неограниченное количество раз
- **Drag & drop в таймлайне** — переупорядочивание элементов маршрута перетаскиванием
- **Кнопки добавления** в маршрут: «+ Отель» и «+ Место» (через модальное окно со списком)
- **Добавление доступных мест** через клик по месту в правой панели на странице маршрута
- **Автомиграция** данных v2 → v3 при загрузке

### Changed
- Формат маршрутов: `placeIds[]` заменён на `items[]` — каждый элемент содержит `{ placeId, duration }`
- `buildRouteUrl` теперь принимает объект маршрута (с `items`) вместо массива `placeIds`
- Первый элемент маршрута по умолчанию — отель (`__hotel__`)
- Обновлён пример данных (`data/example.json`) до версии 3

## [2.0.0] — 2026-05-01

### Added
- Категории мест (eat, walk, see) с фильтрацией
- Поиск мест через геокодер Яндекс.Карт
- Интерактивная карта с маркерами
- Drag & drop сортировка мест внутри категорий
- Импорт/экспорт данных в JSON
- Заметки на день

### Changed
- Добавлено поле `order` у мест для сортировки

## [1.0.0] — 2026-04-15

### Added
- Базовая структура SPA (index.html + ES-модули)
- CRUD для локаций, мест и маршрутов
- Хранение в localStorage
- Стартовые данные из example.json
- Построение ссылки Яндекс.Карт
- Hash-based роутинг
